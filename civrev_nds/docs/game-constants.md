## Game Mechanics (from ARM9 strings)

### Building Bonuses
- `+1 Culture in each city`
- `+1 Production in each city`
- `+1 Science in each city`
- `+2 Production in each city`
- `+2 Science in each city`
- `+2 Trade in each city`
- `+5 Gold in each city`
- `2x city gold production`
- `4x city gold production`
- `2x science in city`
- `4x science in city`
- `Add one population to each city`

### Combat Bonuses
- `+100% city attack`
- `+100% city defense`
- `+100% defend if with other units`
- `50% combat bonus`
- `+1 attack for military units`

### Victory Conditions (from strings)
- **Domination**: Capture all enemy capital cities (palace capture)
- **Technology**: Build space station → Alpha Centauri; OR discover Atomic Theory/Railroad (quick win)
- **Cultural**: 5 Great People + Wonders, then build United Nations
- **Economic**: Accumulate gold, then build World Bank
- 5-turn warning before game ends (highest score = domination victory)
